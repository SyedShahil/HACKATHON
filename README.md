# Swasthya AI

An AI-powered healthcare triage system designed to assist rural communities in **early risk detection and medical decision guidance**.  
The system analyzes **user symptoms (via text or voice)**, predicts potential health risks, and provides **intelligent recommendations** on the next course of action.

This project was built as part of **AADHYA 3.0**, a student-driven hackathon organized by the **Department of CSE, RGUKT RK Valley**.

# 📌 Problem Statement
### AI-Based Early Risk Detection & Intelligent Triage System

Rural communities often lack **structured pre-screening systems** and **immediate access to medical professionals**.

Delayed diagnosis and lack of early triage can lead to severe health complications.

This system aims to bridge the gap between rural patients and healthcare guidance by providing:

- Early risk detection
- Symptom-based analysis
- AI-powered recommendations
- Explainable results

---

# 🚀 Features

## 🧠 AI-Based Symptom Analysis
- Accepts symptom input via **text or voice**
- Supports **multiple symptoms**
- Uses prediction logic to estimate health risks

## 📊 Risk Scoring System
Generates a **risk score between 0 – 100**

| Score Range | Risk Level |
|-------------|-----------|
| 0 – 25 | 🟢 Low |
| 26 – 50 | 🟡 Moderate |
| 51 – 75 | 🟠 High |
| 76 – 100 | 🔴 Critical |

---

## 💡 Intelligent Recommendations

Based on predicted risk:

- 🏡 **Self Care**
- 📞 **Teleconsultation**
- 🏥 **Hospital Visit**
- 🚑 **Emergency Care**

---

## 🔎 Explainable AI
The system explains **why the prediction was made**.

Example:

> "High fever + chest pain + breathing difficulty indicates a high risk of respiratory infection."

This improves **trust and transparency**.

---

## 🎤 Voice Input Support
Users can enter symptoms using **speech-to-text**, making the system accessible for:

- Elderly users  
- Illiterate users  
- Rural populations  

---

## 🌐 Regional Language Support
Supports regional languages for better accessibility.

Examples:
- Telugu
- Hindi
- English
